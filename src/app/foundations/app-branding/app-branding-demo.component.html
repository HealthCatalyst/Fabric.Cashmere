App Title Generator

Last updated February 15, 2024
Overview

The App Title Generator is a tool to create the application title followed just after the Health Catalyst icon in the navbar. For example "Cashmere" in the top navbar of this page is the application title.

Use the configuration options below to make your own application title image:

Theme: Primary Normal Text: Secondary Bold Text:

The produced image is transparent. Use the theme appropriate theme for the backdrop the image will be placed on.
