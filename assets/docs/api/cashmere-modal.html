close(_result?: U)

Closes the modal with an optional result. Use this close method when opening a modal using a component. When doing so, ActiveModal must be injected in order to get access to the close method. To dismiss a modal that was created from a TemplateRef, use the dismiss method on HcModal

Additional interfaces

ModalOptions

Properties
Name Description

autoFocus: boolean

Whether the first focusable element should be focused on open. Defaults to false.

closeIcon: boolean

Adds a close icon to the right side of the header that dismisses the modal Defaults to false.

container: HTMLElement

Optional. Specify a different HTML element to append the modal overlay and modal window. If not specified the modal elements will be added to the body.

data: unknown

Optional. Specify data that will be available on the active modal context.

disableFullScreen: boolean

Defaults to false. When set to true, the modal will no longer display in full-screen for mobile devices.

ignoreEscapeKey: boolean

Defaults to false. Set to true to disable the closure of a modal by pressing the escape key.

ignoreOverlayClick: boolean

Defaults to false. Set to true to disable the closure of a modal by clicking on the overlay.

isDraggable: boolean

Defaults to false. When set to true, a drag handle is added to the top-right corner which can be used to reposition the modal.

isResizable: boolean

Defaults to false. When set to true, a resize handle is added to the bottom-right corner which can be used to adjust the dimensions of the modal. Browser support - https://caniuse.com/mdn-css_selectors_-webkit-resizer

restoreFocus: boolean

Whether the modal should return focus to the previously focused element after closing. Defaults to true.

size: ModalSize

Optional size of the modal window. Defaults to auto sizing based on content. Options are: auto, sm, md, lg, and xl.

tight: boolean

Defaults to false. Set to true to reduce padding of modal sections and title font size.
