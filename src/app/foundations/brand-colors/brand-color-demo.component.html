Overview

The colors included on this page represent our brand. For colors to be used in software applications, please refer to UI Colors. That color set shares the same foundations but is specifically tailored towards values that improve usability for buttons, links and other user interface elements.
Primary Color Group

These are the foundational colors of the brand. The primary color group should carry the most weight in all consumer-facing communications.
Secondary Colors

The secondary color groups, while holistically secondary, can be used as the main color treatment in a supporting graphic or layout, as long as they are paired with our logo and follow the other guidelines of our brand style. The secondary colors are split into two groups to help guide the gradient color points and create harmonious sub- palettes for designers to work with.

Supporting Neutrals

The supporting neutrals are intended primarily for utilitarian applications and web. Black should only beused for text, and the lightest gray should only be used for backgrounds.
