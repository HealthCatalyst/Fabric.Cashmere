Module Import

import {MeasurableModule} from '@healthcatalyst/cashmere';

Services

MeasurableService

This service provides the logic to fill up a container with components contained inside Measurable Components.

Methods
fillContainer(components: MeasurableComponent[], availableSize: number, moreSize: number, direction: "horizontal" | "vertical" = 'horizontal', showFunc: (component, hideFunc: (component)

Fills a container with as many components as will fit. See usage for more details.
Parameter Detail

components: MeasurableComponent[]

The array of components to try and fill the container.

availableSize: number

The available width or height in the container to fill with the given components.

moreSize: number

The width or height of a more control.

direction: "horizontal" | "vertical" = 'horizontal'

Which way to fill the container, either horizontally or vertically.

showFunc: (component

Additional actions to take on a component that is shown. Defaults to a function that does nothing.

hideFunc: (component

Additional actions to take on a component that is hidden. Defaults to a function that does nothing.
Returns
Set<string>

A set containing all of the keys of components that were hidden.

Additional classes

MeasurableComponent

Helper component for creating a container which automatically overflows components into a more menu.

Properties
Name Description

height: number

Calculates and returns the measured height of this container. The height is cached for future calls to this property.

itemKey: string

The unique key of the item displayed inside this measurable component.

width: number

Calculates and returns the measured width of this container. The width is cached for future calls to this property.
Methods
hide()

Hides the contents of this container, forcing a change detection cycle to run.
reset()
show()

Shows the contents of this container, forcing a change detection cycle to run.
