Module Import

import {NavbarModule} from '@healthcatalyst/cashmere';

Additional classes

NavbarComponent

The navbar is a wrapper that positions branding, navigation, and other elements in a concise header.

Properties
Name Description

appIcon: ''

Url to application logo image file

brandIcon: string | HcIcon

Either url to brand icon image file or HcIcon object for a font glyph

fixedTop: false

Fixes the position of navbar to the top of the page. Default is false.

homeUri: unknown[] | string

Router link triggered when home icon is clicked

user: ''

Display name of current user
Methods
refreshNavLinks()

Forces a recalculation of the navbar links to determine how many should be rolling into a More menu. Call this if you've updated the contents of any navbar links.

NavbarLinkComponent

Primary navigation links

Properties
Name Description

active: boolean | null

Forces active state. Overrides router state when set; to remove the override set to null. Default is null.

exact: false

Sets the RouterLinkActive options to match the url exactly to set active state. Default is false. See https://angular.io/api/router/RouterLinkActive#description

linkText: string

The text to display. This can also optionally be the contents within the element

uri: string

RouterLink uri. See https://angular.io/api/router/RouterLink
Methods
hide()

Disable visibility of component from view
show()

Enable visibility of component from view

NavbarCobrandComponent

Element to add a co-branding image to the right of the navbar

Properties
Name Description

color: 'light' | 'dark'

Sets the background for co-branding; use light for images that require a white background. Defaults to light

src: string

Defines the path for the co-branding image. The image should be in landscape (horizontal) format; it will automatically be scaled to fit the navbar.
