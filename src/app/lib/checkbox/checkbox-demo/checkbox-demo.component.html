Checkbox

Checkboxes allow the user to make a binary choice, i.e. a choice between one of two possible mutually exclusive options.

Standard
Standard Checkbox
Disabled
Disabled Checkbox


Checkbox Usage
Checkboxes are typically used with labels, but can be used without in certain situations (table columns, etc).



Angular Component

          
<hc-checkbox name="name">Checkbox Label</hc-checkbox>


Component Parameters

name Type: String Required identifier used to connect the checkbox and label
[disabled] Type: Boolean Boolean value that enables/disables the checkbox
[(model)] Type: Boolean Boolean value for two way binding to the checkbox
