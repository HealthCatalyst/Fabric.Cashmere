Module Import

import {InstanceSwitcherModule} from '@healthcatalyst/cashmere';

Additional classes

InstanceSwitcherComponent

hc-instance-switcher is used to manage multiple instances of the same page.

Properties
Name Description

added: EventEmitter<never>

Empty event emitted when the add instance button is clicked.

chipMaxWidth: 'none'

Configure the max width for the instance names. Default to fit content or none; use maxWidth to constrain and truncate the content to a fixed width (e.g. 150px)

closable: boolean

Controls whether or not to display the close button on the Instance Switcher. Defaults to true.

closed: EventEmitter<never>

Empty event emitted when the close button is clicked on the instance switcher.

editable: false

Controls whether or not the instance chips are editable. Defaults to true.

edited: EventEmitter<IInstance>

instances: IInstance[]

The instances to display on the Instance Switcher. Required.

isOpen: boolean

Sets the Instance Switcher to either open or closed. Defaults to true. Required if closable is also true.

removed: EventEmitter<string>

Value emitted when the close button is clicked on an instance tab. The value is the unique key assigned to the instance.

selected: EventEmitter<string | null>

Value emitted when an instance tab is clicked. The value is the unique key assigned to the instance.

selectedKey: string | null

Sets which instance is selected.

tooltipText: IInstanceSwitcherTooltipText

Configures the tooltip text that shows up on the various controls.
Methods
refreshInstances()

Recalculates which instances should be shown, and which ones should be moved to the more menu.

Additional interfaces

IInstance

Represents an instance to be displayed on the Instance Switcher.

Properties
Name Description

displayText: string

The text that should be displayed on the instance switcher for this instance.

instanceKey: string

The unique key corresponding to this instance.

tooltipText: string

Text displayed for a tooltip on the instance tab

IInstanceSwitcherTooltipText

Configuration for what text to show on tooltips attached to various controls in the Instance Switcher.

Properties
Name Description

addText: string

The tooltip text for the add button.

closeText: string

The tooltip text for the close button.
