About

Select one of many options from a dropdown. This is a styled version of the native <select> element. Component may be disabled if not applicable to a certain context.

Angular Component
            
<hc-select formControlName="city" disabled="false" required="true" valid="true" placeholder="Select a city:">
    <label>Location:</label>
    <option [value]="1" [selected]="value === 1">First</option>
    <option [value]="2" [selected]="value === 2">Second</option>
    <div hcSelectErrors>
        Error messages are added here
    </div>
</hc-select>
Properties
placeholder Type: String Optional string of text to appear before selection is made
disabled Type: Boolean Boolean value that enables/disables the component
valid Type: Boolean When true, highlights the border red and makes any items with the hcSelectErrors directive visible
required Type: Boolean Adds a red asterisk to any labels in the component; form values should still be set to required individually
Directives

hcSelectErrors
Should be added to an element or group of elements that contain error messages to be displayed when the component's valid flag is set to true.

Template Forms

The Select Component is compatible with both template forms and reactive forms in angular. To interact with template forms use [(ngModel)] on the component. Remember to import the FormsModule into your application.

Reactive Forms

To interact with reactive forms specify a formControlName on the component. Remember to import the ReactiveFormsModule into your application.
