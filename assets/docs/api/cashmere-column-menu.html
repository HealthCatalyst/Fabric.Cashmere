Module Import

import {ColumnMenuModule} from '@healthcatalyst/cashmere';

Additional classes

ColumnMenuComponent

Component for toggling and reordering columns within a data grid

Properties
Name Description

allHidden: boolean

Returns true if every hideable column is currently hidden.

cacheKey: ''

Key used when caching the column order and display settings in local storage. If not provided, these settings will not be cached.

colSelectionForm: FormGroup

Collection of the sortable/hidable column forms.

displayedColumnChange: EventEmitter<string[]>

Every time column settings are changed, emits an array of displayed columns in their proper order.

displayedColumns: string[]

An array of the currently displayed column keys sorted in their proper order.

dynamicCols: Array<HcDynamicColumn>

Collection of columns that can be hidden and/or reordered.

menu: HcPopComponent

Reference to the popover menu.

onMenuIconClick: EventEmitter<void>

Fires when button to open menu is clicked.

staticPrefixCols: string[]

Keys for columns that cannot be moved or hidden and sit at the start of the row.

staticSuffixCols: string[]

Keys for columns that cannot be moved or hidden and sit at the end of the row.
Methods
clearAll()

Hide all columns
close()

Close the column menu
columnDropped(event: CdkDragDrop<any, any, any>, isInDataGrid: boolean = true)

Handle rearanging of columns via drag and drop
Parameter Detail

event: CdkDragDrop<any, any, any>

contains data from the drag and drop event

isInDataGrid: boolean = true

if true, we need to take in account the fact that some columns may currently be hidden
open()

Open the column menu
saveCachedPreferences()

Save ordered list of columns with their display status to local storage
selectAll()

Show all columns

true if this property should be shown default. only applicable if the column is hidable.

property name from data element

title: string

ui-ready label for this data element
