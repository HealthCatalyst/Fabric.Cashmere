Drawer Usage
Drawers are great if you need to have content which doesn't fit in a toolbar



Angular Component

          
<hc-drawer-container>
  <hc-drawer>
    Drawer Content
  </hc-drawer>
  Main Content
</hc-drawer-container>


Component Parameters for hc-drawer

mode Type: 'over' | 'push' (default: 'push') Optional - overlay or push content
align Type: 'left' | 'right' (default: 'left') Optional - alignment of the drawer.
opened Type: Boolean (default: false) Optional - whether the drawer starts out opened/closed

Drawer Container Note

An error will be thrown if there are more than two drawers in hc-drawer-container or if both drawers have the same alignment
