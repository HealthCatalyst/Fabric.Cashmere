Elements

h1. Page Title

 <h1></h1> or <div class="h1"></div>
  • Font-size 2.571rem or 36px
  • Line Height 2.625em
  • Weight 600
  • Color $text - $offblack

h2. Subtitle

 <h2></h2> or <div class="h2"></div>
  • Font-size 1.571rem or 22px
  • Line Height 1.5em
  • Weight 400
  • Color $dark-blue

h3. Section Title

 <h3></h3> or <div class="h3"></div>
  • Font-size 1.429rem or 20px
  • Line Height 1.5em
  • Weight 300
  • Color $gray-500

h4. Mini Header

 <h4></h4> or <div class="h4"></div>
  • Font-size 1.2rem or ~17px
  • Line Height 1.313em
  • Weight 300
  • Color $text - $offblack

p. Paragraph

 <p></p> or <div class="p"></div>
  • Font-size 1.143rem or 16px
  • Line Height 1.25em
  • Weight 300
  • Color $text - $offblack
a. Anchor <a href="..."></a>
  • Font-size inherit
  • Line Height inherit
  • Weight inherit
  • Color $primary-brand - $blue
em. Emphasised <em></em>
  • Font-size inherit
  • Line Height inherit
  • Weight inherit
  • Color $gray-500
strong. Strong importance <strong></strong>
  • Font-size inherit
  • Line Height inherit
  • Weight 600
  • Color inherit
code. Code <code></code>
  • Font-size inherit
  • Line Height inherit

  • Color $magneta

Sass Text Utilities

fontSize sass mixin 
    @import '../node_modules/cashmere/sass/cashmere';
.some-class {{ '{' }}
    @include fontSize(16px);    
{{ '}' }}

    @import '../node_modules/cashmere/sass/cashmere';
