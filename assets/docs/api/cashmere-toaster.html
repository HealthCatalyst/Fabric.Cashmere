HcToasterService

Toasts provide users with instant feedback on actions they've taken. For more general information, use a hc-banner.

Methods
addToast(toastOptions?: HcToastOptions, toastContent?: ToastContentType<T>, componentSetup?: ComponentSetup<T>)

Displays a new toaster message with the settings included in toastOptions. toastContent can be used to create entirely custom toasts, but only if the type in toastOptions is set to custom. Be sure to set border-radius: 5px in the style of your custom content template so it matches the toast container. If your custom toast is using a component, the componentSetup parameter accepts an object or function to configure that component.

Additional interfaces

HcToastOptions

Properties
Name Description

body: string

The body text to be displayed in the toast message.

clickDismiss: boolean

When set to true, the user may click the Toast to dismiss it and a close icon is added to the right side for standard types. Defaults to false.

hasProgressBar: boolean

When set to true, a progress bar is added to the bottom of the toast. The default is false. There is a corresponding 0-100 progress value on the hc-toast-ref that may be set to make this a determinate progress bar. If a progress value is not set, the progress bar will be indeterminate.

header: string

A string of header text to be included in the toast message.

image: string | HcIcon

A custom image for the toast that overrides the default for the type. Accepts either a URL for the image or a HcIcon object.

position: string

Position and sizing of the toaster message. Defaults to bottom-right. Options are: top-right, top-center, top-left, top-full-width, bottom-right, bottom-center, bottom-left, bottom-full-width.

timeout: number

Timeout value in milliseconds sets the amount of time the toast is displayed. Defaults to 5000. A value of 0 means the toast will not auto-dismiss.

toastClicked: () => unknown

Function to be called when a click is detected anywhere in the body of the toast

toastClosed: () => unknown

Function to be called when the toast is closed either via timeout or click

type: string

Sets the background color of the toast. Defaults to success. Options are: success, info, warning, alert, custom

width: number

Width of the toast in pixels. Minimum is 300px. Or, pass in 0 for an unconstrained width. Defaults to 300px.
