Data Tables

Last updated Apr 15, 2019
Basic Example

Add the base class .hc-table to an HTML <table> to get the default Catalyst table stylings. Set the alignment of a cell using .hc-table-justify-left (default), .hc-table-justify-center, or .hc-table-justify-right. Justify right is typically used for numerical values.

First Name Last Name MRN Birth Date Payor
Samuel Walker Z1223334444 Mar 27, 1978 MetLife
Whitney Allen Z5223335555 Jan 14, 1954 New York Life
Kendra Jacobs Z6223336666 Aug 27, 1990 Prudential Financial
Gerald Lynch Z7223337777 Mar 15, 1981 MetLife 
        
    <table class="hc-table">
        <thead>
            <tr>
                <th>First Name</th>
                <th>Last Name</th>
                <th>MRN</th>
                <th>Birth Date</th>
                <th>Payor</th>
            </tr>
        </thead>
        <tbody>
            <tr>
                <td>Samuel</td>
                <td>Walker</td>
                <td>Z1223334444</td>
                <td>Mar 27, 1978</td>
                <td>MetLife</td>
            </tr>
            <tr>
                <td>Whitney</td>
                <td>Allen</td>
                <td>Z5223335555</td>
                <td>Jan 14, 1954</td>
                <td>New York Life</td>
            </tr>
            ...
        </tbody>
    </table>
Sorting Indicators

Add .hc-col-sortable to a <th> so that it appears sortable.

  • Use .hc-active-sort to designate the currently sorted column
  • Use .hc-sort-asc and .hc-sort-desc to designate sort direction
  • For a left aligned sort indicator, use .hc-col-sortable-left

Sortable Active Sort (Ascending) Nope Active Sort (Descending) Left indicator
Samuel Walker Z1223334444 Mar 27, 1978 MetLife
Whitney Allen Z5223335555 Jan 14, 1954 New York Life
Kendra Jacobs Z6223336666 Aug 27, 1990 Prudential Financial 
        
    <table class="hc-table">
        <thead>
            <tr>
            <th class="hc-col-sortable">Sortable</th>
            <th class="hc-col-sortable hc-active-sort hc-sort-asc">Active Sort (Ascending)</th>
            <th>Nope</th>
            <th class="hc-col-sortable hc-active-sort hc-sort-desc">Active Sort (Descending)</th>
            <th class="hc-col-sortable-left">Left aligned</th>
            </tr>
        </thead>
        ...
    </table>

Selected Row

Add .hc-row-selected to a <tr> so it appears selected.

Also, you can add .hc-action-row to a <tr> or .hc-action-table to a <table> to get a blue hover effect on rows.


First Name Last Name MRN Birth Date Payor
Samuel Walker Z1223334444 Mar 27, 1978 MetLife
Whitney Allen Z5223335555 Jan 14, 1954 New York Life
Kendra Jacobs Z6223336666 Aug 27, 1990 Prudential Financial 
        
    <table class="hc-table hc-action-table">
        ...
        <tr class="hc-row-selected">...</tr>
        ...
    </table>
