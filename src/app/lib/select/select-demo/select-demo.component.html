Select

Menus display a list of choices with one choice per line.

Standard Select Component
Disabled Select Component



Select Types
There are two primary types of dropdown menus used in Cashmere: one is an extension of the button component, and the other is this select component. The guidelines for which dropdown to use is based off what items are contained in the dropdown. The button dropdown is for a list with action items (e.g. delete all selected items). If the dropdown contains a list of selectable items, then this component should be used.

Template Forms
The Select Component is compatible with both template forms and reactive forms in angular. To interact with template forms specify [(ngModel)]= on the component. Remember to import the FormsModule into your application.

Reactive Forms
To interact with reactive forms specify formControlName= on the component. Remember to import the ReactiveFormsModule into your application.



Angular Component

          
<hc-select formControlName="city" [disabled]='false' placeholder="Select a city:"
    [options]="['Philadelphia', 'Atlanta', 'Salt Lake City', 'Chicago', 'Orlando']">
</hc-select>


Component Parameters

placeholder Type: String Optional string of text to appear before selection is made
[options] Type: Array An array of strings for the select options
[disabled] Type: Boolean Boolean value that enables/disables the button
