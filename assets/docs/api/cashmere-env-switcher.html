Module Import

import {EnvSwitcherModule} from '@healthcatalyst/cashmere';

Additional classes

EnvSwitcherComponent

Properties
Name Description

canOpenInNewTab: boolean

If true, will show an icon that allows user to click to open an environment in a new tab. When clicking this icon, a (openInNewTab) event will be fired that the consuming app will need to listen for and respond to. Defaults to false.

canSelectMultiple: boolean

If true, will allow multiple environments to be selected at once. Defaults to false.

disabled: boolean

When disabled, the environment switcher cannot be opened. Defaults to false.

environmentOptions: IMetadataEnvironment[]

The metadata environments to show in the switcher.

failedToLoad: false

Set to true if unable to load environments from MDS. A "Failed to load" message will be shown in the switcher. Defaults to false.

listHeader: "Environments"

Header to display above the expanded list of environments. Defaults to 'Environments'.

loading: false

Set to true while waiting for environments to load from MDS. Defaults to false.

openInNewTab: EventEmitter<IMetadataEnvironment>

Event that fires when the "open in new tab" action is triggered. Outputs the selected environment. Consuming app will need implement functionality to actually open a new tab with the given environment active.

pop: HcPopComponent

Reference to the popover containing the environment selection menu.

reloadEnvironments: EventEmitter<void>

Event that fires when users click a link to reload environments. This link is shown if failedToLoad is set to true.

selectedEnvironments: IMetadataEnvironment[]

The currently active metadata environments. NgModel/FormValue is bound to just the environment IDs, but you can access this property for an array with the entire environment model.

updateEnvironments: EventEmitter<Array<IMetadataEnvironment>>

Event that fires when model is updated. Outputs an array of the currently selected environments. Consuming app will need to set appropriate environment headers refresh elements on page as needed.
Methods
closeMenu()

Manually close the app switcher menu.
openMenu()

Manually open the app switcher menu.
setActiveEnvironments(envs: number[])

Convenience method for setting the current active environment(s) by their Ids. Invalid environment Ids will not be selected.

Additional interfaces

IMetadataEnvironment

Properties
Name Description

color: string

Hex code for the color associated with an environment. Used in the environment badge ui element. Color options include: "#ffffff", "#7acb91", "#00acac", "#00a859", "#e7c447", "#ebba82", "#f89012", "#f13c45", "#eea2C0", "#a94c9d", "#3194fe", "#ace5ff". If no color (or an invalid color) is provided, will default to white (#ffffff).

description: string

Describes the environment.

id: number

A unique identifier for the environment.

name: string

The full name of environment.

shortName: string

An abbreviated environment name. Used in environment badge ui element.

tenantCode: string

An identifier representing a given tenant. (For example, a particular client.)
