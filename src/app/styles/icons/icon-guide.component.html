Overview

The purpose of Health Catalyst’s icon set is to provide a standard visual language around concepts frequently used in our applications. For an optimal user experience, any time elements like data marts, bindings, or columns appear, they should be represented the same way – regardless of the application.

This icon set focuses primarily on items related to healthcare data. It is meant as a complement to UI glyph libraries like Font Awesome, not a replacement. Note that Cashmere does not have a dependency on any icon font. Any glyphs that are used in the components have been base64 encoded into the component’s styles.
