Vertical Tabs

Vertical tabs ease the burden of large pages by dividing them into easy to digest sub-pages

Example
Tab content 1 Tab content 2 Tab content 3



Vertcal Tab Usage
Vertical Tabs are most often used as a form of secondary navigation on pages that have a large number of sub-pages. For those with a fixed, smaller number of sub-pages, using Horizontal Tabs might be more appropriate. Horizontal and Vertical tabs may be used in conjunction on pages where secondary and tertiary levels of navigation are required.



Angular Component

        
            <hc-vertical-tabs>
            	<hc-vertical-tab title="Tab 1">
            		Tab content 1
            	</hc-vertical-tab>
            	<hc-vertical-tab title="Tab 2">
            		Tab content 2
            	</hc-vertical-tab>
            </hc-vertical-tabs>


Component Parameters

title Type: String Input used to change the title of the tab
