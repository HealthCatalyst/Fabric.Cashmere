Module Import

import {SidenavModule} from '@healthcatalyst/cashmere';

Additional classes

SidenavComponent

Component for standard sidebar navigation

Properties
Name Description

activeTabKey: ''

Key of the tab that should be marked as active.

canToggle: true

If true, sidebar can toggle between collapsed and expanded. Will have an expand/collapse button. Defaults to true.

collapsed: false

If true, collapse the sidebar to a minimal width. Defaults to false.

collapsedChange: EventEmitter<boolean>

Event triggered when the sidebar is expanded or collapsed.

collapsibleChildren: boolean

True if nested children should be hideable. Overrides openChildrenByDefault and openChildrenWhenActive. Defaults to true.

containerCssClass: ''

Add a custom css class to the sidebar container.

darkMode: false

True for a darker sidebar navigation. Defaults to false.

favoriteClicked: EventEmitter<SidenavLinkClickEvent>

Event triggered when a favorite is clicked.

favoriteItemIconClicked: EventEmitter<SidenavLinkClickEvent>

Event triggered when the action icon for a favorite list item is clicked. Typically for "unfavoriting" an item.

favorites: SidenavLink[]

Collection of objects representing user favorited items.

favoritesDescription: 'Mark favorites to pin them here.'

Description of the favorites section for display in the UI as a tooltip

favoritesItemActionIconClass: 'hc-ico-star-filled'

A clickable action icon to show next to each favorite list item. Default is a filled yellow star.

favoritesItemActionIconTooltip: 'Remove from favorites'

Tooltip for a favorite's action icon.

favoritesTitleIconClass: 'hc-ico-star'

An icon to show in the favorites header.

favoritesTitleLabel: 'Favorites'

Label for the favorites section.

isLoadingFavorites: false

True to show loading indicator for favorites.

isLoadingTabs: false

True to show loading indicator for tabs. Use while data is loading to show tabs

openChildrenByDefault: boolean

True if nested children should be shown by default. Defaults to false.

openChildrenWhenActivated: true

True if nested children should be shown if the link is active. Overrides openChildrenByDefault. Defaults to true.

routerLinkActiveClass: ''

If using routerLinks, routerLinkActiveClass can be used to set appropriate class for the active route. If used, the activeTabKey property will be ignored. If a class besides active-tab is used, you'll need to provide your own css.

showFavorites: true

True to show scrollable favorites section.

showTreeLines: true

True to show lines of tree structure. Defaults to true.

tabClicked: EventEmitter<SidenavLinkClickEvent>

Event triggered when a tab is clicked.

tabGroupHeaderClicked: EventEmitter<SidenavLinkClickEvent>

Event triggered when a tab group header is clicked.

tabGroups: SidenavTabGroup[]

Collection of tabGroups. These are styled like the main tabs, but are grouped together under a separate header.

tabs: SidenavLink[]

Collection of objects representing the main links in the sidenav.

width: '260px'

Set width of the sidebar. Include unit. Can be pixels, rem, em, etc. Default is 260px
Methods
close()

Collapse the sidebar to a minimal width.
collapseAllNestedLinks()
open()

Expand the sidebar to full width.
openAllNestedLinks()
toggle()

Toggle the sidebar's expand/collapse state.
toggleAllNestedLinks(shouldOpen: boolean = true)

SidenavLink

Properties
Name Description

badgeColor: 'slate-gray-200'

Select a valid cashmere color for the badge background color (from the primary, secondary, or neutral palette. eg 'blue', 'slate-gray-300', or 'light-pink')

badgeHTML: string

Text or markup to be inserted into a badge. WARNING: You are responsible to sanitize any unsafe user input.

children: SidenavLink[]

Nested sub links. Will be displayed in a tree structure

clickable: true

If false, this link will not respond to clicks. Will not look disabled, but will just serve as a non-clickable label. Defaults to true.

description: string

Describes the link. Will be used for hover tooltips.

disabled: false

If true, this link will not respond to clicks and will look disabled. Defaults to false.

iconClass: string

Icon associated with this link

key: string

Unique identifier for this link. If not provided at initialization, will be auto generated.

labelHTML: string

Serves as the primary label in the UI. Will override "title". WARNING: You are responsible to sanitize any unsafe user input.

onClick: (e: MouseEvent, link: SidenavLink) => void

Instead of a route, can provide a callback function. Alternative is to listen to SideNav's tabClicked or favoriteClicked @Output events.

open: false

True if this parent link is open and children links should be shown.

routerLink: string

Passed to routerLink

subText: string

Extra detail for this link, preferably less than 10 characters. Shown as a sub label in the UI.

tabCSSClass: string

Additional CSS class to apply

title: string

Name of the link. Used as the primary label in the UI.

SidenavHeaderDirective

Apply this attribute directive to any element to have it projected into hc-sidenav as a header.

SidenavBodyDirective

Apply this attribute directive to any element to have it projected into hc-sidenav as the body.

SidenavFooterDirective

Apply this attribute directive to any element to have it projected into hc-sidenav as the body.
