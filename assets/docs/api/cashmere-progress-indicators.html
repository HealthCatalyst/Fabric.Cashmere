Module Import

import {ProgressIndicatorsModule} from '@healthcatalyst/cashmere';

Additional classes

ProgressDotsComponent

Properties
Name Description

color: ProgressDotsColor

Use light on darker backgrounds and dark for lighter backgrounds.

isCentered: true

If true, the loader will center itself within its container.

isMini: false

If true, you'll get a teeny tiny little loader.

ProgressSpinnerComponent

Properties
Name Description

color: string

Color of the spinner: blue, green, purple, orange, red, gray, white

diameter: number

Set the diameter of the circle, in pixels. Minimum is 20, maximum is 250.

hasChannel: true

If true, include background "channel" circle.

isCentered: true

If true, the spinner will center itself inside its container.

isDeterminate: false

If true, switches to determinate mode. Must pass in progress (0-100%), instead of having the loader spin freely.

progress: number

(0-100%) Only used if "isDeterminate" is set to true.

ProgressBarComponent

Properties
Name Description

color: string

Color of the bar: blue, green, purple, orange, red, gray, white

isDeterminate: false

If true, switches to determinate mode. Must pass in progress (0-100%), instead of having the loader spin freely.

label: string

Optional text string to be displayed beneath the progress bar

progress: number

(0-100%) Only used if "isDeterminate" is set to true.

showValue: false

If true, displays the current percentage of the progress bar and expands the height to accomidate. Only used if "isDeterminate" is set to true.
