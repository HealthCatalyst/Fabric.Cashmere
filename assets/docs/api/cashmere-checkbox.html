Event emitted whenever the state changes

Whether the checkbox is checked.

disabled: boolean

Whether the checkbox is disabled.

id: string

Unique id for the checkbox element. If none is supplied, one will be auto-generated.

Whether the checkbox is indeterminate. It can represent a checkbox with three states.

If true, the checkbox is for display purposes (not user interaction). As such its checked/unchecked state can only be controlled programatically. Useful for embedding in an ng-select typeahead
