Additional classes

PaginationComponent extends BasePaginationComponent

The pagination control enables the user to navigate across paged content. Although commonly used with tables and data grids, this control may be used any place where paged data is used.

Properties
Name Description

displayWidth: 'lg' | 'md' | 'sm'

Sets the controller to a specific width type - lg, md, or sm. Typically adjusted in a window resize listener for responsive layouts. Defaults to lg.

hidePageSize: boolean

Whether to hide the page size selection UI from the user. Defaults to false.

isFocused: false

length: number

The total number of items to be paged through

page: EventEmitter<PageEvent>

Event emitted when the paginator changes the page size or page index.

pageNumber: number

The currently displayed page. Defaults to 1.

pageNumberChange: EventEmitter<number>

Emits the new page number when the page number changes.

pageSize: number

Number of items to display on a page. Defaults to 20.

pageSizeChange: EventEmitter<number>

Emits the new page size when the page size changes.

pageSizeOptions: number[]

The set of provided page size options to display to the user. Defaults to [10, 20, 50].

tight: false

If true, remove all default padding and margin. Defaults to false.

totalPages: number

The computed total number of pages

LoadMorePaginationComponent extends BasePaginationComponent

A simple "load more" pagination button.

Properties
Name Description

buttonStyle: string

Sets style of button. Choose from: 'primary' | 'primary-alt' | 'destructive' | 'neutral' | 'secondary' | 'link' | 'link-inline'. Defaults to 'secondary'.

buttonText: 'Load more'

Text to show on the button. Defaults to 'Load more'.

length: number

The total number of items to be paged through

page: EventEmitter<PageEvent>

Event emitted when the paginator changes the page size or page index.

pageNumber: number

The currently displayed page. Defaults to 1.

pageNumberChange: EventEmitter<number>

Emits the new page number when the page number changes.

pageSize: number

Number of items to display on a page. Defaults to 20.

pageSizeChange: EventEmitter<number>

Emits the new page size when the page size changes.

totalPages: number

The computed total number of pages

PageEvent

Change event object that is emitted when the user selects a different page size or navigates to another page.

Properties
Name Description

length: number

The current total number of items being paged

pageNumber: number

The current page index

pageSize: number

The current page size

previousPageNumber: number

Index of the page that was selected previously
