Radio Button Usage

Radio buttons are used in groups and always include labels.

Radio buttons can be used with ngModel in reactive or template-driven forms.

When enabled, only one radio button in the group should be set to checked.

A default button can be set to checked when the form is initialized (either manually or via ngModel).
