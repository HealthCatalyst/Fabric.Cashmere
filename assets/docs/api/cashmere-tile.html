Additional classes

TileComponent

Container element to help segment content visually against a gray background. The tile will expand to the height and width of the content it contains.

Properties
Name Description

tight: boolean

If true, compress the default padding in the tile. Defaults to false

TileHeaderDirective

Applies one of the Cashmere standard tile header stylings to an element

Properties
Name Description

type: TileHeaderType

Sets the styling of the header, choices include blue, center, and title; defaults to blue
