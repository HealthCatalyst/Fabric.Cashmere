About

Displays a row of items that can have multiple lines per item with icons. Useful in a variety of situations including menus, filter lists, etc.
Example

Snow

SnowFlake line 1

SnowFlake line 2

Another SnowFlake

Address

Address Line 1

Address Line 2
Angular Component
            
        <hc-list>
            <h3 hcListSubheader>Snow</h3>
            <hc-list-item>
                <hc-icon fontSet="fa" fontIcon="fa-snowflake-o" hcListIcon></hc-icon>
                <h4 hcListLine>SnowFlake line 1</h4>
                <span hcListLine>SnowFlake line 2</span>
            </hc-list-item>
            <hc-list-item>
                <hc-icon fontSet="fa" fontIcon="fa-snowflake-o" hcListIcon></hc-icon>
                <h4 hcListLine>Another SnowFlake</h4>
            </hc-list-item>
            <h3 hcListSubheader>Address</h3>
            <hc-list-item>
                <img src="assets/CashmereLogo.svg" hcListAvatar>
                <h4 hcListLine>Address Line 1</h4>
                <span hcListLine>Address Line 2</span>
                <hc-icon fontSet="fa" fontIcon="fa-plus"></hc-icon>
            </hc-list-item>
        </hc-list>
Components

hc-list

This is the container for the hc-list-items

hc-list-item

Acts as a host element for the individual items found in a hc-list-item. Represents an item within the hc-list-items

Directives

hcListIcon

Positions the element to the left and styled as an icon. Should be used in conjunction with hc-icon

hcListAvatar

Positions the element to the left and gives it a circular border

hcListLine

This is the main text content of the hc-list-item. Add this directive to content you want to show up in the middle content section

hcListSubheader

Gives a list section a header
