Code Display Guidelines

Last updated Jan 30, 2019
Overview

Source code often needs to be included within app documentation or within apps themselves. The following are guidelines on how to style code used in these instances, both inline or within larger code blocks. The primary goal of code styling is differentiate it from the surrounding body copy and ensure it is easily readable. Both types of code have a background-color set to $block-text-background .

Elements
Inline code <code></code>
  • Font-size 1rem
  • Line Height 140%
  • Weight 400
  • Color $magenta 
Code blocks
 <pre><code></code></pre>
  • Font-size 1rem
  • Line Height 140%
  • Weight 400
  • Color $text
