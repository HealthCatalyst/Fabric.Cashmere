Module Import

import {ResizableModule} from '@healthcatalyst/cashmere';

Additional classes

ResizableComponent

A container that the user can resize the dimensions of

Properties
Name Description

disabled: boolean

Removes the resizable handle from the container when set to true

max: number

The maximum value (in pixels) that the container may expand to. There is no max by default.

min: number

The minimum value (in pixels) that the container may contract to. There is no min by default.

position: ResizablePosition

Determines which corner of the container the drag handle will be available on. Accepts bottom, left, right, or top.

resized: EventEmitter<DOMRect>

Emits a DOMRect after a drag event is complete
Methods
reset()

Resets the resizeable container to its orginal default size

ResizableStaticDirective

Can be added to a sibling of a resizable element that does not need a resize handle

ResizableContainerDirective

Can be added to the parent element containing resizable and static elements

Properties
Name Description

vertical: boolean

Orients the container for horizontal or vertical resizing; defaults to true
