About

Breadcrumbs are a graphical control element used as a navigational aid within our app subheader. It allows users to keep track of their location within the current application. Typically positioned on the left side of the subnavbar.
Example
Responsive Behavior

At phone resolution, the component's media query will remove the navigation tree and display a "Back" button linking to the previous page. If currently at the root of the tree, the current page name will be displayed.
