About

The icon component encapsulates font icons. It provides an abstraction that makes using icons easier and more uniform. Another benefit is that component developers can "select" the icon from the "transcluded" user content, allowing the icon to be placed appropriately.

<hc-icon fontSet="fa" fontIcon="fa-snowflake-o"></hc-icon>
Component Parameters
fontSet Type: String Required - font icon family to use. For example to use fontawesome, you need to provide "fa"
fontIcon Type: String Required - font icon to use from the specified font icon family
