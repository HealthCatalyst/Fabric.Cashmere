Additional classes

HcScrollNavComponent

Container for scroll navigation links.

Properties
Name Description

hasDynamicContent: false

Set to true to enable the component to change for dynamic content changes that might not be picked up by Angular

isScrolling: false

scrollNavWithContent: false

Set to true to enable scrolling the nav link pane as the content pane scrolls
Methods
refreshScrollNavLinks(updateScrollLinkArray: ScrollNavLinkDirective[] = [])

Refresh the scroll nav links when dynamic changes have been made. updateScrollLinkArray parameter is optional

HcScrollNavContentComponent

Contains scrollable content that is navigable via hc-scroll-nav links.

Properties
Name Description

bufferSpace: this.DEFAULT_BUFFER

Number in pixels, used to give a little leeway in the shifting of the active nav when scrolling. Defaults to 0. Example: If set to 40, if showing just the bottom 40 pixels of the section before, count the next section as active.

hasDynamicContent: false

Set to true to enable the component to change for dynamic content changes that might not be picked up by Angular

lastTargetMinHeightAdjustment: 0

Adjust the min height of the last target

makeLastTargetFullHeight: true

If true, will force the height of the final scroll target area to be the height of the scrollable container. This is helpful if you want the last target in the content area to be able to scroll to the top. You can alternatively target the last item with css. Defaults to true.

nav: HcScrollNavComponent

Reference to the scroll nav component.

newSectionInView: EventEmitter<string>

Fires when a new section is scrolled into view. Broadcasts the id of that section.

sectionInView: string

Id of the current section scrolled into view.

shouldAnimateScroll: true

If true, applies smooth scrolling via css. Defaults to true.
Methods
checkActiveSection()

Will update the navigation state.
refresh()

Scroll to top and reset the 'automatic full height for the last item' setting.
refreshScrollNavTargets(updateScrollTargetArray: ScrollNavTargetDirective[] = [])

Refresh the scroll nav targets when dynamic changes have been made. updateScrollTargetArray parameter is optional
scrollToTop()

Helper function to scroll to the top of the content area.
setActiveSection(scrollTarget: string)

Sets the active section to passed in scrollTarget

ScrollNavTargetDirective

Marks the host element as a linkable section within an hc-scroll-nav-content. Must set the id and link to it via an hcScrollLink.

Properties
Name Description

nativeElement: HTMLElement

The nativeElement of the corresponding hcScrollTarget that you would like to link to.
