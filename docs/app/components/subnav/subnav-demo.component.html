Subnavbar

Secondary navigation bar appearing below the primary navbar.

Subnavbar Usage

The hc-subnav component is an optional part of the page header, it is not required. It is meant to be a flexible area that can be used as per the needs of the application. Some may choose to use it for breadcrumbs, others for secondary navigation, filtering, search, etc. hc-subnav with fill the width of the container it is placed in. A fixedTop parameter may be set to pin the subnav to the top of the page below hc-navbar.

Angular Component

        
<hc-subnav>
    <div>Content goes here</div>
    <div hcSubnavRight>
        <button hc-button color="primary">Example</button>
    </div>
</hc-subnav>

Directives

hcSubnavRight
Can be added to a div or individual component to align it to the right side of the subnav. Primary action buttons are often placed on the right side of the subnav.

Component Parameters

fixedTop Type: Boolean Statically positions the subnav below the navbar
