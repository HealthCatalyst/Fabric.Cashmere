import {ButtonToggleModule} from '@healthcatalyst/cashmere';

Additional classes

ButtonToggleComponent

Button Toggle components are the individually selectable buttons used within the Button Toggle Group component.

Properties
Name Description

disabled: boolean

Whether the individual toggle is disabled

selected: boolean

Whether the toggle is currently selected. Updates the value of the group when set. Defaults to false.

value: string

The value assigned to this particular toggle. Used by the group to get/set the value of the selected item(s).

ButtonToggleGroupComponent extends HcFormControlComponent

hc-button-toggle-group components are on/off toggles with the appearance of an hc-button. These toggle groups may be configured to behave as single-select (like radio buttons), or multi-select (like checkboxes).

Properties
Name Description

buttonStyle: string

Sets style of toggle. Choose from: 'primary' | 'primary-alt' | 'destructive' | 'neutral' | 'secondary'. If needed, colors from the primary or secondary palette may be used as well (e.g. 'pink', 'red-orange', etc). Defaults to secondary.

disabled: boolean

Whether the entire toggle group is disabled.

multiple: boolean

Whether multiple button toggles can be selected. Defaults to false.

onChange: (value: unknown) => void

onTouch: () => unknown

selectionChangedEvent: EventEmitter<ButtonToggleChangeEvent>

Event fired whenever a change is made to any button toggle in the group.

size: string

Sets size of toggle. Choose from: 'sm' | 'md' | 'lg' |. Defaults to md.

value: unknown

valueRequired: boolean

Whether at least one button must remain selected.
