About

Chips represent complex entities in small blocks, such as filters, contacts, or system information. They may be removed if they display a delete icon. Chips are often grouped in collections, for which a supporting chip-row component may be used.

Chip Usage

The chip module consists of two componenets:

  • hc-chip - the base component
  • hc-chip-row - the optional container for chips
Properties
Chip Component
action Type: Boolean Whether to display a delete button and pointer cursor
color Type: String 'neutral', 'red', 'yellow', 'green' (default='neutral')


Chip Row Component
wrap Type: Boolean If false, constrain the container to one line with overflow ellipses (default=true)
