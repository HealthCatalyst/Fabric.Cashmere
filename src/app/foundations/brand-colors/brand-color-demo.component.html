Health Catalyst Brand Colors

Last updated January 21, 2021
Overview

The colors included on this page represent our brand. For colors to be used in software applications, please refer to UI Colors. That color set shares the same foundations but is specifically tailored towards values that improve usability for buttons, links and other user interface elements.
Primary Colors

These are the foundational colors of the brand. Primary colors must be prominent in all materials.
Secondary Colors

Secondary colors can be applied in different formats to represent categories, break up content, or to add visual interest. Secondary colors may only be used when paired with a primary color or gradient. No more than one secondary color should be used in the same placement.

HC Charcoal provides a nice contrast to the brighter secondary color and can be used more frequently as a background color or anchoring element. HC Purple and HC Magenta should be used more sparingly.
