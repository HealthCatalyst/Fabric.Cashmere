Overview

Source code often needs to be included within app documentation or within apps themselves. The following are guidelines on how to style code used in these instances, both inline or within larger code blocks.

Elements
Inline code <code></code>
  • Font-sizeinherit
  • Line Heightinherit
  • Weightinherit
  • Color$magneta
 
Code blocks
 <pre><code></code></pre>
  • Font-size1rem
  • Line Height140%
  • Weightinherit
  • Color$text
