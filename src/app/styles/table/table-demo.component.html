Basic Example

Add the base class .hc-table to an HTML <table> to get the default Catalyst table stylings.

Set the alignment of a cell using .hc-table-justify-left (default), .hc-table-justify-center, or .hc-table-justify-right. Justify right is typically used for numerical values.

Sorting Indicators

Add .hc-col-sortable to a <th> so that it appears sortable.

  • Use .hc-active-sort to designate the currently sorted column
  • Use .hc-sort-asc and .hc-sort-desc to designate sort direction
  • For a left aligned sort indicator, use .hc-col-sortable-left

Sortable Active Sort (Ascending) Nope Active Sort (Descending) Left indicator

Table Action Icons

For tables that have actions repeated on each row, we recommend showing those actions as icons that appear when hovering over the row. Designate an action icon with .hc-action-ico, and wrap those actions in .hc-table-actions.

We recommend showing at most 3 actions per row, and if there are more, put them in a vertical ellipsis menu (.hc-elip-menu). Each action icon should leverage hcTooltip so that a description of the action is shown in a tooltip on hover.

If a single action icon should always be shown, whether the row is being hovered over or not, use .hc-action-ico-show. For mobile, all actions should always be shown. In that case you can apply .hc-table-actions-show to the container.

Table Bulk Actions

For cases where actions can be performed on multiple rows at once, use .hc-table-bulk-actions. Use buttons with buttonStyle="minimal" size="sm". Rows should be selectable using checkboxes.

Empty Table State

Also available are some helper styles to standarize the look of empty and error state tables (.hc-empty-table-block).

The messaging should be as specific as possible. For example, is the table empty because of an error, because there are no items available at all, or just because searching or filtering returned no results?

If there's a specific action that would help the user along when facing an empty or error state, you should provide it. For example, if no items have been added, you could provide a button to add one.

First Name Last Name Birth Date Payor
No entries to show. 

    <div class="hc-empty-table-block">
        <span class="hc-empty-table-block-ico hc-empty-ico"></span>
        <span class="hc-empty-table-block-msg">No entries to show.</span>
        <button hc-button>Add entry</button>
    </div>
