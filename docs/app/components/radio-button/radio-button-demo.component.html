Radio Button Usage

Radio buttons are used in groups and always include labels.

When enabled, only one radio button in the group should be set to checked.

A default button should be set to checked when the form is initialized.

Angular Component

        
<hc-radio-button [disabled]='false'>Radio Button Label</hc-radio-button>

Component Parameters

[disabled] Type: Boolean Boolean value that enables/disables the radio button
