Navbar

The navbar is a wrapper that positions branding, navigation, and other elements in a concise header.

Navbar Usage

The Navbar consists of five componenets:

  • hc-navbar - the main wrapper component
  • hc-navbar-link - the main navigation links
  • hc-navbar-icon - icons and buttons displayed on the right side
  • hc-app-switcher - include app switcher functionality, if this element is not present, the catalyst flame will revert to navigating to the home page when clicked
  • hc-navbar-menu - the responsive menu triggered when the navbar is used on smaller screens

App Switcher

To include the app switcher, do the following:

<hc-app-switcher></hc-app-switcher>
 
{{moduleSetup}}

Fixed Top

To fix the navigation bar to top of the screen, such that it stays put when your scroll, set the fixedTop property to true. You'll need to also add the following to your body style sheet: 

{{fixedTopNavbar}}

Responsive Menu

Unlike some responsive menus that automatically pull content from the navbar into them, we intentially left ours configurable so that developers can determine what should or shouldn't be included on smaller screens. Use the appSwitcher property to set whether the "View all applications" button is included at the bottom of the menu. hc-list components should be used for the menu options with

separators in between. See the demo code at the top of this page for an example of how to define a responsive menu for hc-navbar.

Component Parameters

Navbar Component
user Type: String Current user
appIcon Type: String Path to svg of app logo
homeUri Type: String RouterLink uri to home page
fixedTop Type: Boolean Statically positions the navbar to the top of the page (See Fixed Top section above)
Navbar Link Component
linkText Type: String The text to display
active Type: boolean (optional) forces active state (default = null)
uri Type: String RouterLink uri
exact Type: Boolean Sets the RouterLinkActive options to match the url exactly to set active state. (default = false)
Navbar Menu Component
appSwitcher Type: Boolean Whether to display the "View all applications" button at the bottom of the menu (default = true)
