Additional classes

HcBannerComponent

Notification banners are used for general information about the state of the application or upcoming events. For instant feedback responding to user actions, use a toaster message.

Properties
Name Description

bannerClose: EventEmitter<MouseEvent>

If the banner can be dismissed, emits when the banner is clicked to close

clickDismiss: boolean

When set to true, a close icon is added to the right side and bannerClose emits on a click. Defaults to false.

type: BannerType

Style of the notification banner; Defaults to info. Options are: success, info, warning, alert

BannerStampDirective

Including hcBannerStamp wraps the enclosed text in a stamp to provide greater emphasis
