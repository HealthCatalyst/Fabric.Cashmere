About

Chips represent complex entities in small blocks, such as filters, contacts, or system information - and they may be removed if they display a delete icon. Chips are often grouped in collections, for which a supporting chip-row component may be used. The most common use case for chips is to display data filters, so a standard filter button is also included which can be dropped into the chip-row.

Chip Usage

The chip module consists of three componenets:

  • hc-chip - the base component
  • hc-chip-row - the optional container for chips
  • hc-filter-button - the optional filter button often used inside a chip-row
Properties
Chip Component
action Type: Boolean Whether to display a delete button and pointer cursor
color Type: String 'neutral', 'red', 'yellow', 'green' (default='neutral')


Chip Row Component
singleRow Type: Boolean If true, constrain the container to one line with overflow ellipses


Filter Button Component
labelOff Type: String Text to display when button is off (default='Filters off')
labelOn Type: String Text to display when button is on (default='Filters on')
buttonState Type: Boolean If true, the button will display its 'labelOn' text
