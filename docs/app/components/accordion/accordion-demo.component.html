Accordion Usage

When you have auxiliary data to show the user that isn't necessary for them to see

Angular Component

            
                <hc-accordion [triggerAlign]="alignment" [hideToolbar]="showToolbar">
                    <hc-accordion-toolbar>Toolbar</hc-accordion-toolbar>
                    Accordion content
                </hc-accordion>
            
            

    import { AccordionModule } from '@healthcatalyst/cashmere';

    @NgModule({
        imports: [
            ...,
        AccordionModule,
        ],
        declarations: [...],
        exports: [..]
    })
    export class SomeModule { }

Inputs

triggerAlign Type:'left' | 'right' (default: 'left') Placement of the toolbar button
hideToolbar Type: Boolean Show/Hide the accordion toolbar
open Type: Boolean Set accordion opened state

Events

opened Accordion is in the open state
openStart Accordion is beginning to open
closed Accordion is in close state
closeStart Accordion is beginning to close
